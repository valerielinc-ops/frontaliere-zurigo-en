Application process

  • You take over the installation and setup of the notebooks provided by Zurich and accompany the IT onboarding of new agencies including the on-site setup of devices, Microsoft 365, MFA (Okta and Microsoft Authenticator), SharePoint, Outlook, Teams, OneDrive and other applications

  • In-depth knowledge of Windows 11, Microsoft 365, Active Directory (AD DS) & Microsoft Entra ID and Network Technology Basics (DNS, DHCP, VPN, TCP/IP)
  • Experience with cloud services (M365, Microsoft Entra ID, Microsoft Intune)
  • Experience with ServiceNow, SharePoint and Confluence as well as ITIL knowledge and IT security awareness
